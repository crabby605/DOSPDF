DOS-in-a-PDF

FreeDOS, emulated end to end, running inside a single PDF file.

An Intel 8086, its BIOS, and a FreeDOS floppy are emulated in JavaScript that executes inside the PDF viewer’s own scripting engine — no server, no plugins, no WebAssembly. Inspired by Linux in a PDF.
