DOS-in-a-PDF

FreeDOS running inside a PDF

I saw Linux running in a PDF and wondered if I could get x86 DOS running too. A few weeks later, here we are. and it's very funny DOS running in a PDf,

It emulates an 8086, BIOS and FreeDOS floppy disk entirely inside PDF JavaScript. No server, no installs, just a PDF.
